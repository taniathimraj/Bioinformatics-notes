# Bioinformatics Notes

Personal reference notes compiled from hands-on experience in computational biology and bioinformatics. These notes cover tools and workflows I use regularly for data analysis on GCP, ATAC-seq, and general compute tasks.

## Contents

| Folder | Topic | Description |
|--------|-------|-------------|
| `GCP/` | Google Cloud Platform | VM management, storage buckets, tmux, running scripts |
| `ATAC-seq/` | HOMER Motif Analysis | Motif enrichment analysis pipeline for ATAC-seq data |
| `Git/` | Git & GitHub | Version control workflow for bioinformatics projects |
| `SSH/` | SSH | Connecting to remote machines, file transfer, SSH keys |

## About

These notes are written for personal reference but shared openly in case they are useful to others. Commands and workflows are tested on Mac (local) and Google Cloud Platform (remote).
